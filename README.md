# Student Management System (Core Java)

A console-based Student Management System built using Core Java.
This application performs CRUD operations and demonstrates OOP principles,
Java collections, and exception handling.

## Features
- Add student
- View students
- Update student
- Delete student
- Exception handling for invalid input

## Technologies Used
- Java 17
- Core Java (OOP, Collections, Exception Handling)
- VS Code

## How to Run
```bash
javac *.java
java Main
